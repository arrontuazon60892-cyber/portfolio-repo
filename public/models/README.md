# Hero robot model

`robot.glb` is the RobotExpressive model by Tomás Laulhé, with glTF modifications by Don McCurdy.

- Source: https://threejs.org/examples/models/gltf/RobotExpressive/
- License: CC0 1.0 Universal
- Upstream file: https://github.com/mrdoob/three.js/blob/dev/examples/models/gltf/RobotExpressive/RobotExpressive.glb

The model is served as a static public asset at `/models/robot.glb`. Keep it in `public/models/` so it is not bundled into Vercel server functions.
